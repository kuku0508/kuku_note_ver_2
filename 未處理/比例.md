比例（proportion, $p$），是指一個整體中，各個部分與整體之間的比值關係。
$$
p=\frac{各部分數量}{整體數量}
$$

例如我們可以看到政治人物A的支持者男女數量分別為：

|     | 男            | 女            |     |
| --- | ------------ | ------------ | --- |
| 數量  | 700          | 300          |     |
| 比例  | 700/1000=0.7 | 300/1000=0.3 |     |

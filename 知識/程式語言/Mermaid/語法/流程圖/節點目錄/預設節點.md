當我們今天想要建立一個節點的時候，我們可以直接在`flowchart`中打入任何連著的字，就會有框著你打的字的節點。

```Mermaid
flowchart 
id
```

```mermaid
flowchart
id
```

```Mermaid
flowchart
test
```

```mermaid
flowchart
test
```

```Mermaid
flowchart
為了神皇
```

```mermaid
flowchart
為了神皇
```

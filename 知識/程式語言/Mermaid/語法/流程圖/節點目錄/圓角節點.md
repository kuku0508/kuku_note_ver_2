當我們今天想要改變一下節點的形狀，改成圓角的節點，那我們可以在[[預設節點]]後面打一個小括號，就可以把節點用成圓角節點。小括號裡面可以打任何你想打的字。
```Mermaid
flowchart 
id1( )
```
```mermaid
flowchart
id( )
```
```Mermaid
flowchart
id(圈圈圓圓圈圈)
```
```mermaid
flowchart
id(圈圈圓圓圈圈)
```

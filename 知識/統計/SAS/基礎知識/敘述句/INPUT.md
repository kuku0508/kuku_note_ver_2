INPUT，是在[[datastep]]裡面用來「讀取」或「定義」資料讀入格式的[[SAS敘述句]]。
簡單來說就是告訴SAS我們讀的資料裡面的變數，長什麼樣子，告訴SAS讀的辨識不同變數的規則是什麼。
- - -
# 基本語法
```SAS
INPUT 變數名稱1($) 位置或格式1 變數名稱2($) 位置或格式2 ......;
```

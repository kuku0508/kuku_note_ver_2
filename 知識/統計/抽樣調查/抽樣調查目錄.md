# 簡單隨機抽樣
```base
views:
  - type: table
    name: Default view
    order:
      - file.name
    filters:
      and:
        - file.inFolder("知識/統計/抽樣調查/簡單隨機抽樣")
        - file.folder == "知識/統計/抽樣調查/簡單隨機抽樣"
display:
  file.name: Name
```

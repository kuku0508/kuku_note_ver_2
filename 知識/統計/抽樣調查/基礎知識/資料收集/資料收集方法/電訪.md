電訪（Telephone Interview），是一種[[資料收集方法]]，是指訪員透過電話與受訪者進行訪談，依照訪問大綱或是問卷築堤提問並記錄回答。
- - -

| 優點✅    | 缺點❌              |
| ------ | ---------------- |
| 成本低    | 無法觀察非語言行為        |
| 執行效率高  | 訪談時間通常無法太長       |
| 匿名性較高  | 被拒絕訪問機率高         |
| 不受地理限制 | 對對話速度、語調與清晰度要求較高 |
|        | 僅能使用對話說明，無其他輔助   |

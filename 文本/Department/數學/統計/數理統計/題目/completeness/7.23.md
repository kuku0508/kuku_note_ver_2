# 問題：
If $az^2+bz+c=0$ for more than two values of z, then $a=b=c=0$.
Use this result to show that the family $\lbrace b(2,\theta):0<\theta<1\rbrace$ is complete.
# 回答：
let g be a function such that $E(g(x))=0$ ,$\forall \theta \,\,that\,\, 0<\theta<1$
$$
\begin{align}
E[g(x)]&=\sum^2_{i=1}g(x)\cdot C^2_x\theta^x(1-\theta)^{(2-x)}\\
&=g(x)\cdot C^2_0\theta^0(1-\theta)^{(2-0)}+\\
&g(x)\cdot C^2_1\theta^1(1-\theta)^{(2-1)}+\\
&g(x)\cdot C^2_2\theta^2(1-\theta)^{(2-2)}\\
&=g(x)\cdot1\cdot1(1-\theta)^2+g(x)2\cdot\theta(1-\theta)+
\end{align}
